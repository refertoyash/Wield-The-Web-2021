What You'll learn

HTML

HTML is the standard markup language to display document in a web browser.It stands for "Hypertext Markup Language"

CSS

Cascading Style Sheets is a style sheet language used for describing the presentation of a document

Javascript

JS is a light-weight object-oriented programming language which is used by several websites for scripting the webpages

Bootstrap

Bootstrap is a free and open-source CSS framework directed at responsive, mobile-first front-end web development.

Git and Github

Git track the changes of source code.GitHub is a service for hosting.

FAQs

  • Is there any entry fee to be a part of this event?

    No, the event is free of cost.

  • How much time will I get to submit the website?

    You’ll get at least 4 – 5 days to submit the website.

  • Do I have to make a completely dynamic website?

    You need to make a template that is a single-page website.

  • How many rounds are there in the event?

    There will be two rounds in the event. The first one will include web development classes along with daily quizzes. The second one will include template making.

  • What will be the overall marking scheme of this event?

    Your overall rank will be judged on these two bases.

    ROUND-1

    1) Your performance in daily quizzes.

    ROUND-2

    2) You have to create and submit a template to prove your learning.

    TOTAL - 100 marks

  • Why should we participate in this event?

    Participating in such events improves your web development skills and helps you to explore new technologies.

  • How many questions will the quiz comprise? Which topic will they be?

    In your daily quizzes, each day there will be only 10 MCQ-based ques on the topic you have been taught on that particular day.

  • I do not know much about Git and Github. Can I still participate?

    Yes, of course. You don’t need to worry about it, our team will organize a dedicated session on Git and GitHub.
